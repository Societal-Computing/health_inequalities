The country dataset contains the same variables as the subnational datasets plus some additional country level variables.
See the 'data_readme.txt' file for details on the variables that are also available subnationally.

Below are the country level variables available for the country datasets.

# - - - - - Country level indicators - - - - -

Variables with country level data for the country to which the gadm unit belongs.



* Variables from the Global Gender Gap Report
	"score_gggi_2021_cntry": The Global Gender Gap score
	"EconOpp_ggg_subidx_2021_cntry": The Economic opportunities gender gap subindex
	"EducAttn_ggg_subidx_2021_cntry": The Educational Attainment gender gap subindex
	"Health_ggg_subidx_2021_cntry": The Health gender gap subindex
	"PoliticEmpow_ggg_subidx_2021_cntry": The political empowerment gender gap subindex

* Variables from the Digital Gender Gaps website:
	"dgg_Internet_GG_OnlineModel_pred_Jan2019_cntry": Online model predicted Internet GG for Jan 2019
	"dgg_Mobile_GG_OnlineModel_pred_Jan2019_cntry": Online model predicted Mobile GG for Jan 2019
	"dgg_Internet_GG_OnlineModel_pred_May2021_cntry": Online model predicted Internet GG for May 2021
	"dgg_Mobile_GG_OnlineModel_pred_May2021_cntry": Online model predicted Mobile GG for May 2021
	
* World Bank variables:
	"MobCell_subscr_per100ppl_cntry": Mobile Cellular subscription per 100 people (https://data.worldbank.org/indicator/IT.CEL.SETS.P2)
	"GDP_per_capita_current_USD_cntry": GDP per capita (https://data.worldbank.org/indicator/NY.GDP.PCAP.CD)
	"Indiv_using_Internet_percOfPop_cntry": Individuals using the interent as % of the population (https://data.worldbank.org/indicator/IT.NET.USER.ZS)
